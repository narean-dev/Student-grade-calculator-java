Student Grade Calculator

A simple and efficient tool to calculate student grades based on marks obtained in various subjects. This project helps automate the process of finding totals, averages, and assigning letter grades.

🚀 Features

Input Marks: Enter marks for multiple subjects.Total & Percentage: Automatically calculates the sum and the mean percentage.Grade Assignment: Assigns grades (A, B, C, or fail) based on pre-defined logic.Validation: Basic checks to ensure marks are within the valid range (0-100).
